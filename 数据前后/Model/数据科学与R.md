# 数据预测与R
## 预测分类
+	时间分类：中长短
+	假设条件：条件预测、非条件预测
+	预测结果：定性预测、定量预测、定时预测
+	趋势是否确定：确定性、非确定性预测
+	预测的依据：动态预测、静态预测
##  预测基本原则
+	目的性
+	连贯性：时间、结构连贯（规律稳定）
+	关联性：中心化关联、类比化关联
+	近大远小
+	概率性
+	反馈性
+	经济性

##	大数据
+	volume:体量大
+	variety:种类多
+	value:价值密度低
+	velocity:速度快
## 大数据分析
+	全样非抽样
+	效率非精确
+	相关而非因果

## 预测流程
+	确定主题
	+	内容：
		+	明确预测的主体和指标
		+	预测的周期多长，达到什么样的精度

	+	要素：
		+	指标，主体，精度，周期，用户，成本，数据
+	收集数据

		+	产品参与对象和活动：客户类，产品类，营销类，物流类，财务类，服务类

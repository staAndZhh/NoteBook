##	uwsgi配置nginx
1.	Request > Nginx > uWSGI > Django > uWSGI > Nginx > Response
2.	请求先交由Nginx，如果是静态内容就自己处理了，如果是动态内容就交给uWSGI服务器，uWSGI服务器处理整个Django项目的Python代码，响应请求，原路返回，但是与fastcgi不同，Nginx、uWSGI和Django可以独立部署，然后整合。

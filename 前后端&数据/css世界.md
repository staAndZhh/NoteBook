#	概念
##	选择器
+	类	.
+	ID	#
+	属性  [xx = "xx"]{}
+	伪类	  first-child:xx	:last-child
+	伪元素	::first-line	::first-leter
+	后代		空格
+	子选择		>
+	兄弟		~
+	相邻		+
+	特殊规则 @ 	@media

##	html类型

	+	div li table

##	Tips:
+ 	子元素100%时，父元素必须要有具体的高度
###	属性
+	float :属性定义元素在哪个方向浮动。
+	clear	:元素的哪边上不允许出现浮动元素
+	white-space	:设置如何处理元素内的空白
